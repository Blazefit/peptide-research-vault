# Project: Blaze Peptide Research Vault

Build a public static HTML/CSS/JS site for CrossFit Blaze/Jason based on extracted public peptide research data.

Rules:
- Original design inspired by research dashboards; do not copy Reptides UI/branding/assets.
- Must be mobile-first and work on a phone.
- Static only: no build step required for GitHub Pages.
- Use data/peptide_details.json as the source.
- Include educational/research-only disclaimer. No medical advice, no purchasing instructions.
- Make it searchable/filterable by tier and source type.
- Keep it fast and self-contained enough for GitHub Pages.
